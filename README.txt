# rpg-java

RPG-Java est un jeu où le but est de survivre pour espérer arriver à la fin..
Pour changer de niveau, vous devez tuer tous les ennemies de la carte(ils peuvent se téléporter et arriver dans La maison) et prendre le portail.

Vous pouvez fabriquer toutes sortes d'outils et d'armes(il vous faudra d'abord débloquer la recette en ayant au moins une fois les items nécessaires dans l'inventaire), vous permettant de faire différentes actions(comme casser un arbre, miner de la pierre, taper des ennemies...).
Certaines interactions ne peuvent se faire que lorsque le bon objet est sélectionné(casser un arbre avec une hâche, miner de la pierre avec une pioche..).

Vous pouvez vous soigner avec des aliments(le plus commun étant la pomme).


[Commandes]:


 Déplacements:
  Z aller vers le haut
  Q aller vers la gauche
  S aller vers le bas
  D aller vers la droite

 Inventaire:
  Molette vers le haut pour faire défiler l'inventaire du joueur vers la droite
  Molette vers le bas pour faire défiler l'inventaire du joueur vers la gauche
  
 Marchant/Inventaire de fabrication:
  E pour l'inventaire de fabrication
  F pour interagir avec les PNJ
  Clique gauche pour accepter l'échange(avec un PNJ/inventaire de fabrication)

 Actions:
  Clique gauche pour attaquer
  Clique droit pour interagir(lorsqu'il y a une indication)
  Echap pour mettre en pause le jeu
